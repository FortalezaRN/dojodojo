# dojodojo

## Install

```
$ npm i -g dojodojo
```

## Configuration

You should configure a dojodojo.json in the folder that you wanna
start a dojo session with the participants informations like this:

```json
{
    "participants": [
        {
            "username": "Henri Cavalcante",
            "email": "hnri_mxel@hotmail.com",
            "active": true

            "active": true

```

## Usage

```
$ dojodojo
```
